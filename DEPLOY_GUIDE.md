# تعليمات رفع الموقع على الاستضافة (GHPages, Netlify, Vercel)

لقد قمت بتجهيز الموقع بالكامل ليكون جاهزاً للنشر. إليك الخطوات البسيطة لرفعه:

### 1. تجهيز الملفات (Build)
الموقع مبني باستخدام **React + Vite**. عند رفعه على استضافة مثل **Netlify** أو **Vercel**، ستحتاج فقط لربط حساب GitHub الخاص بك وسيقوم الموقع بالبناء تلقائياً.

- **Command:** `npm run build`
- **Output Directory:** `dist`

### 2. أهم الإعدادات
- **اللغة:** تم ضبط الموقع ليدعم اللغة العربية (`dir="rtl"`) بشكل افتراضي في محركات البحث.
- **التوافق:** الموقع متوافق تماماً مع الموبايل وجميع الشاشات.
- **التواصل:** جميع الأزرار (واتساب، اتصال) تعمل مباشرة بمجرد الضغط عليها.

### 3. خيارات الاستضافة المقترحة:
1. **Netlify (الأسهل):** قم بسحب مجلد `dist` بعد تشغيل أمر البناء، أو اربط المستودع مباشرة.
2. **Vercel:** ممتاز جداً لمشاريع React، يدعم البناء التلقائي بمجرد رفع الكود.
3. **Hostinger / CPanel:** إذا كنت تستخدم استضافة عادية، قم بتشغيل `npm run build` محلياً ثم ارفع محتويات مجلد `dist` إلى مجلد `public_html`.

### 4. تحسين محركات البحث (SEO)
لقد أضفت الكلمات المفتاحية والعناوين المناسبة للدكتور محمود الشيوي لضمان ظهور الموقع بشكل احترافي في نتائج البحث ومواقع التواصل الاجتماعي.
